# Notification Sounds

Place the following audio files in this directory:

- **notification.mp3** - Default notification sound (Info notifications)
- **success.mp3** - Success notification sound
- **warning.mp3** - Warning notification sound
- **error.mp3** - Error notification sound

## File Requirements
- Format: MP3, OGG, or WAV
- Duration: 1-3 seconds recommended
- File size: < 100KB each

## Free Sound Resources
You can download free notification sounds from:
- https://notificationsounds.com/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/

## Fallback
If sound files are not found, notifications will still work but without audio feedback.
The system will gracefully handle missing sound files.
